# go-gorilla-websocket-example

Example of implementing WebSocket using Go + Gorilla

## Development Environment

* Apple MacBook Pro (13-inch, M1, 2020)
* macOS Sonoma 14.2.1 (23B81)
* go@1.21.4
* Visual Studio Code

## Install

```shell
go get -u github.com/gorilla/websocket
```

## Run

```shell
make go-run
```
