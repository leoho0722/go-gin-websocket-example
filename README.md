# go-gin-websocket-example

Example of implementing WebSocket using Go + Gin

## Developement Environment

* macOS Sonoma 14.1.1 (23B81)
* go@1.21.4
* Visual Studio Code

## Install

```shell
go get -u github.com/gin-gonic/gin
go get -u github.com/gorilla/websocket
```

## Run

```shell
make go-run
```
