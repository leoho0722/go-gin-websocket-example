# go-x-net-websocket-example

Example of implementing WebSocket using Go + x.net

## Developement Environment

* Apple MacBook Pro (13-inch, M1, 2020)
* macOS Sonoma 14.1.1 (23B81)
* go@1.21.4
* Visual Studio Code

## Install

```shell
go get -u golang.org/x/net/websocket
```

## Run

```shell
make go-run
```
